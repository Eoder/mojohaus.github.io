Welcome to the MojoHaus Project (previously known as Mojo@Codehaus)

IMPORTANT NOTICE: if you arrived on this page because you wanted to access some plugin documentation, please be aware that as per the recent Codehaus Shutdown (Starting June 2015), we're still working on infra aspects of the migration and not all documentations are available yet and anything not found is fallbacking here.

Neither is this website up-to-date. Hopefully, most of the active plugin documentations will be back in the upcoming weeks, and we'll also update the current site asap. Sorry for inconvenience. Please ping us at the new mailing list if needed. -- The MojoHaus Team

The MojoHaus project is a collection of plugins for Apache Maven 2 & 3.

Releases of the plugins are available from the Maven Central repository. Snapshots are available on request through the users mailing list or by emailing the maintainer. Each plugin has a list of people involved in creating or supporting it.

Plugins

For information on the plugins hosted at the Mojo project see the plugins list. Detailed information and instructions for each plugin can be found on its Maven site.

You can also check out other plugins from the subversion repository and build them yourself for general usage, including the sandbox plugins. See using sandbox plugins for information about building from source and retrieving sandbox snapshot releases.

Getting Support

If you have problems or concerns with plugins in the Mojo project please contact the author, ask on the users mailing list or join us on IRC. There are also instructions on how to report an issue should you run into one.

Development

If you would like to contribute to the project, either through fixing an existing plugin or by donating a new plugin, contact us on the development mailing list or join us on IRC.
